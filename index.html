Esta página web tiene 3 objetivos. El primer objetivo es informar al lector sobre datos científicos de los osos pandas. En este tipo de investigación siempre se dejarán las fuentes de donde se obtuvieron los datos para que el lector pueda verificar la fiabilidad de la información dada. El segundo objetivo es que el lector pueda conocer algunas películas, series y juegos relacionadas o que contengan a los osos panda ya sea como personaje principal o secundario. Este punto principalmente con el fin de entretener y no tanto de informar. El tercer objetivo es presentarle al lector algunas organizaciones las cuales ayudan a los osos panda para que el lector sepa qué acciones se están realizando para evitar la extinción de este animal y para que pueda donar a esta causa si así lo desea.
